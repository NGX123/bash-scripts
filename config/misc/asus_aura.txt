### ON WINDOWS ###
## Aura ##
1. Download asus aura(not armory crate)
  1. Laptop Model(https://rog.asus.com/us/Laptops/ROG-Zephyrus/ROG-Zephyrus-M-GM501-Series/) -> Downloads -> Aura sync -> Download the aura sync from the link on the bottom and not the armory crate
  2. https://dlcdnets.asus.com/pub/ASUS/mb/14Utilities/Lighting_Control_1.07.79_V2.2.zip
2. Extract the Lighting_Control_1.07.79_V2.2.zip
3. Go to folder with files extracted from Lighting_Control_1.07.79_V2.2.zip
4. Run AsusSetup.exe, install aura and reboot

## Armory Crate ##
1. Download Armory Create
  1. Laptop Model(https://rog.asus.com/us/Laptops/ROG-Zephyrus/ROG-Zephyrus-M-GM501-Series/) -> Downloads -> Aura sync -> Click on the large download button and check if it is Armory Crate
  2. https://dlcdnets.asus.com/pub/ASUS/mb/14Utilities/ArmouryCrateInstallTool.zip
2. Extract ArmouryCrateInstallTool.zip
3. Got to folder with files extracted from ArmouryCrateInstallTool.zip
4. Run ArmoryCrateInstaller.exe and run it
